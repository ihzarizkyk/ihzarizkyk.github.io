Cara Buat Profile Readme Github

Github merupakan platform website yang terkenal bagi para Developer, Programmer, Pegiat IT di seluruh dunia. Baru-baru ini github merilis fitur baru pada bulan juli 2020 yaitu Github Profile Readme. Dengan adanya fitu tersebut, anda bisa menghias profile github menjadi lebih kreatif dan menarik.

Nah, pada artikel kali ini saya akan membahas mengenai Cara Buat Profile Readme Github simak berikut ini :

  1. Buka profile github anda, sebagai contoh akun saya github.com/ihzarizkyk dan buat repo baru

  1. buat nama repo dengan nama yang sama dengan nama akun github anda dan visibilitas public serta centang Add a README file, contoh : github.com/ihzarizkyk/ihzarizkyk

  1. Silahkan edit file README sesuai dengan keinginan Anda dan jika sudah klik commit

  1. Terakhir, akses kembali akun github anda. contoh akun github saya : github.com/ihzarizkyk

Nah, sampai disini anda sudah berhasil membuat profile github dengan file README. Mudah bukan? selamat mencoba…
