Deploy Hugo Ke Github

Github memiliki layanan hosting untuk website statis yang bernama github pages. Nah, nantinya file hasil render dari folder public hugo akan di repository github pages. (Membuat Folder Public Hugo)

Begini Caranya :

1. Buat Repository dan beri namawebsite.github.io yang nantinya akan disambungkan dengan github pages. pastikan Visibilitynya Public, karena jika private anda harus upgrade langganan pro.

2. Buka pages.github.com dan klik Github repository. setelah itu anda akan diarahkan ke github.com untuk memilih repo yang telah dibuat pada langkah pertama tadi.

3. Buka Terminal atau CMD dan masuk ke direktori tempat project anda disimpan dan juga folder public.

cd project
cd public

4. Inisialisasi git :

git init

5. Tambah Repository Remote dari repo yang telah dibuat diawal (HTTPS) :

git remote add origin https://github.com/<username>/<namawebsite.github.io>.git

6. Upload

git add .
git commit -m "upload pertama"
git push -u origin master

Dengan begitu semua file dan folder didalam folder public akan terupload ke repo yang telah anda buat.
