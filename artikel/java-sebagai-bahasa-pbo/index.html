Java Sebagai Bahasa Pemrogaman Berbasis Objek

Java adalah Bahasa Pemrograman Yang Cukup Tua dan Populer. Java merupakan Bahasa Pemrograman berbasis objek, berbeda dengan C yang paradigmanya pemrogramannya prosedural. Dengan mempelajari Bahasa Pemrograman Java, Maka secara tidak langsung kita sudah dibiasakan menggunakan Paradigma Pemrograman Berbasis Object. Java didesain oleh James Gosling dan diciptakan Sun Microsystems yang sebelum bernama Java, biasa disebut OAK. Nama Java dipilih atau terinpirasi karena James Gosling Menyukai Kopi yang berasal dari Pulau Jawa, Maka dari itu dinamakan Java. Saat ini Bahasa Pemrograman Java sudah mencapai Versi 8.0 sejak 2014 dan 2 tahun belakang sudah mencapai Versi 11.0.8

Kenapa Menggunakan Java ?

  • Sederhana dan Berbasis Objek
  • Karena Java merupakan bahasa pemrograman cross platform. Dimana dengan sekali mengetikkan kode, sudah bisa dijalankan diberbagai Platform (Linux, Unix, Windows)
  • Bahasa Pemrograman Populer (Ranking 2 Versi TIOBE Index October 2020)
  • Bahasa Pemrograman yang banyak digunakan oleh Institusi Pendidikan untuk proses pembelajaran pemrograman
  • Banyak dipakai oleh Perusahaan Enterprise
  • Bahasa Pemrograman Tingkat Tinggi (Bahasa Pemrograman yang mendekati Bahasa Manusia)

Cara Kerja Bahasa Pemrograman

  • Komputer hanya mengenal 0 dan 1
  • Lalu, dengan adanya bahasa pemrograman bisa menjadi perantara antara Manusia dan Komputer agar komputer bisa bercakap melakukan instruksi tertentu
  • Karena komputer hanya mengenal angka 0 dan 1. Bahasa Pemrograman perlu compiler atau interpreter agar bisa mengubah sekumpulan perintah dari bahasa pemrograman ke dalam angka 0 dan 1 yang dimengerti oleh Komputer.

Compiler : Mengkompilasi kode menjadi file yang bisa dieksekusi

Interpreter : Mengkompilasi kode dan Menjalankan file secara langsung

Cara Kerja Java (Compiler & Interpreter)

  • File .java dicompile dengan Javac lalu dijadikan file .myclass
  • File .myclass dieksekusi dengan Java interpreter dan dijadikan file program yang siap dijalankan

Jenis – Jenis Paradigma Pemrograman :

Paradigma adalah gaya pemrograman yang berhubungan dengan bagaimana sebuah masalah bisa dipecahkan dan ditemukan solusi terbaiknya.

  • Functional Programming : paradigma pemrograman yang program tersebut dibangun dengan sekumpulan fungsi – fungsi.
  • Procedural Programming : paradigma pemrograman yang dimana menggunakan instruksi kode yang terurut dan terstruktur
  • Object Oriented Programming : paradigma pemrograman yang dimana nantinya program akan dibangun dengan banyak objek dan disatukan dalam class
