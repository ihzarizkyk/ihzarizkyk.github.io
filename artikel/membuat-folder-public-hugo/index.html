Membuat Folder Public Hugo

Folder Public merupakan folder hasil render dari file-file markdown Hugo yang nantinya akan digunakan untuk deploy ke hosting.

Setelah semua proses build website statis Hugo. Maka, saatnya untuk deploy ke hostingan agar website statis kalian bisa dilihat semua orang alias online. Kalian bisa upload semua file dan folder yang ada di dalam Folder Public.

Lalu Bagaimana Cara Membuat Folder Public ?

1.Masuk Ke direktori website hugo yang telah kalian buat

cd <YOUR PROJECT>

2.Buka Terminal atau CMD Kalian, pastikan telah berada di direktori project kalian

3.Ketikkan berikut :

hugo -D

4.tekan enter dan folder public dari project kalian akan selesai dibuat.
