Cara Kontribusi Open Source di Github

Github merupakan hal yang tidak asing bagi para Developer, Programmer dan Pegiat IT dari seluruh dunia. Github digunakan para programmer dari seluruh dunia untuk menaruh kodenya didalam Repository. Selain itu juga Github digunakan untuk menggarap Proyek Open Source bersama-sama seperti linux, hugo, laravel dan masih banyak lagi proyek open source lainnya.

Nah, Artikel kali ini saya akan membahas bagaimana cara berkolaborasi menggarap Proyek bersama Developer atau Programmer lainnya.

Alat - alat yang dibutuhkan :

  • Git (VCS)
  • Akun Github
  • Text Editor

Langkah - langkah berkontribusi dalam proyek open source di github :

1.Baca Aturan Kontribusi :

biasanya pada setiap proyek open source di github itu disertakan tata cara berkontribusi. Aturannya biasanya ditulis didalam file CONTRIBUTING.md dan untuk menyertakan nama kontributor di file CONTRIBUTORS.md

Anda harus mengikuti aturan yang telah dibuat untuk berkontribusi. Jika tidak anda akan ditolak ketika mengajukan kontribusi di Github

2.Fork dan Clone Repository Proyek Open Source :

Selanjutnya adalah melakukan Fork yang berarti anda menyalin repository orang lain ke dalam repository anda.

dan selanjutnya anda bisa clone repository tersebut ke dalam local computer. clone merupakan mendownload file repository ke dalam komputer lokal

lalu, buka folder hasil clone dari repository di Github ke dalam text editor. Pada kasus kali ini saya mengedit perubahan untuk tema hugo-vitae

setelah sudah merubah file diatas, silahkan save dan buka terminal atau CMD untuk mengirim perubahan ke repo yang ada di Github.

git add . (untuk semua file)
git add filename.md

commit perubahan yang anda lakukan :

git commit -m "melakukan penambahan pada filename.md (misal)"

push ke dalam repo yang di fork tadi :

git push -u origin master

masukkan username dan password akun github anda

3.Selanjutnya melakukan pull request :

Pull Request adalah meminta untuk menggabungkan kode yang telah anda rubah dengan repository yang aslinya.

Klik Pull request di Repo yang anda telah fork tadi. lalu klik New pull request

Jika tidak ada tabrakan antar kode, akan ditampilkan pesan Able to merge dan selanjutnya klik Create pull request

Silahkan isi judul dan deskripsi pull request yang akan kamu lakukan :

dan klik Create pull request

Jika sudah akan tampil seperti ini :

Anda tinggal menunggu tanggapan dari pemilik repo proyek open source tersebut. bisa diterima atau tidak tergantung dari keputusan pemilik repo tersebut.
