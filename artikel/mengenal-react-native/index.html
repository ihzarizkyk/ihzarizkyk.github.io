Mengenal React Native

Apa itu React Native?

Pada mata kuliah ini kita akan mempelajari React Native. React Native adalah Framework Javascript yang didevelopt oleh Tim Facebook yang dimana anda bisa membuat kode program yyang bisa dijalankan layaknya native android tapi ditulis dalam bahasa javascript. React Native mempunyai slogan Learn Once, write anywhere yang artinya dengan hanya menulis kode program di reac native, anda bisa menerapkannya untuk os ios dan android.

Kenapa Menggunakan React Native?

Alasan mengapa menggunakan React Native adalah antara lain:

  1. Populer
  2. Banyak dipakai oleh berbagai perusahaan dunia, kecil maupun besar.
  3. Learn Curva yang pendek jika sudah mempelajari javascript
  4. Terpercaya
  5. Selalu update karena dikembangkan oleh Facebook
  6. dan lainnya…

Siapa yang menggunakan React Native?

Banyak Perusahaan yang menggunakan React Native, diantaranya adalah Instagram, facebook, ruangguru, microsoft dan masih banyak lagi.

Core Component dan Native Component

Mengenal Core Component:

adalah tampilan yang digunakan untuk membuat dasar block dari UI yang dimaksud seperti elemen kotak, menampilkan text, gambar, menerima inputan dan lainnya

Mengenal Native Component:

Tampilan yang digunakan di android atau ios menggunakan react components. Ketika render, react components akan menampilkan layaknya platform ios atau android.

React Fundamentals

Dalam React Fundamentals, anda akan mempelajari mengenai :

  1. Components : block element yang dipasang di kode react native dan didalam class atau function

  2. JSX : syntax yang terdapat didalam javascript untuk menuliskan semisal parameter atau arguments di function dan class

  3. Props : properti data pada sebuah komponen, jadi sebuah props adalah data(state) yang bisa di akses di komponen lain (Properties Components). Jadi tidak seperti state, props dapat di akses di komponen lain. Sederhananya Props adalah State yang dikirim ke komponen lain

  4. State : event didalam kode jsx ata react yang nantinya bisa menampilkan feedback atau reaksi balik jika dijalankan atau data yang berubah - ubah didalam components atau data sementara.
