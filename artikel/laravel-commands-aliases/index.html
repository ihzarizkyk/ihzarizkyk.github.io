Create Laravel Command Aliases on Linux

Bagaimana cara menjalankan perintah yang dipakai dalam Laravel secara singkat?

Anda masih menuliskan perintah berikut ini?

  • php artisan serve
  • php artisan make:controller HomeController
  • php artisan make:controller HomeController –resource
  • php artisan make:model Post -m

Kenapa tidak menggunakan perintah berikut?

  • pas (php artisan serve)
  • pm HomeController (php artisan make:controller HomeController)
  • pmr HomeController (php artisan make:controller HomeController –resource)
  • pmm Post (php artisan make:model $* -m)

Bingung gimana cara buatnya?

Oke, langsung saja kita buat.

  1. Pertama, buka terminal dan ketikkan perintah sudo nano ~/.bashrc
  2. Kedua, Save dan exit
  3. Ketiga, ketikkan sudo nano ~/.bash_aliases
  4. Keempat copy perintah shortcut berikut :

alias pas=“php artisan serve” alias pc=“php artisan make:controller $* " alias pcr=“php artisan make:controller $* –resource” alias pmm=“php artisan make:model $* -m”

  1. Kelima, save dan exit
  2. Keenam, ketikkan source ~/.bashrc
  3. Ketujuh, restart terminal linux kamu
  4. Kedelapan, masuk ke direktori project laravelmu dan coba ketikkan perintah shortcut yang sudah kamu buat seperti diatas.

Selesai dan Semoga Bermanfaat
