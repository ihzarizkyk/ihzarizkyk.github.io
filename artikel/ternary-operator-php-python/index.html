Ternary Operator PHP dan Python

Dalam Pemrograman ada kalanya kita harus menulis kode yang ringkas dan efisien dalam penggunaannya. Nah, dalam pemrograman untuk menyingkat penulisan kondisi if…else… adalah dengan menggunakan Ternary Operator.

Ternary Operator Adalah Operator yang berfungsi untuk menyingkat kode kondisi if…else… menjadi 3 bagian. Akan tetapi, setiap bahasa pemrograman memiliki aturan tersendiri untuk penulisan ternary operator, dibawah ini adalah aturan umum penulisan ternary operator dalam Bahasa Pemrograman PHP :

<?php

$variable = condition ? statement true : statement false;

Implementasi :

<?php

$nilai = (10>9) ? "true" : "false";
echo $nilai;

// Output : true

Tanpa Ternary Operator :

<?php
if(10>9)
{
	echo "true";
}else{
	echo "false";
}

Untuk aturan penulisan Ternary Operator dalam Bahasa Pemrograman Python sebagai berikut :

variable = statement true if (condition) else statement false

print(variable)

Implementasi :

nomor = "benar" if (10>9) else "salah"
print(nomor)

# Output : benar

Tanpa Ternary Operator :

if(10>9):
	print("benar")
else:
	print("salah")

# Output : benar

Nah, kurang lebih begitu contoh dari Penggunaan Ternary Operator dalam Bahasa Pemrograman PHP dan Python. Semoga Bermanfaat
